description: 業界×業務から新ツールの機会を1枚 issue 化する（opportunity-scout に委譲）
argument-hint: <業界> <業務>
---
`opportunity-scout` に委譲して、次の機会を backlog の issue に整形してください。
- 業界/ペルソナ: $1
- 業務（面倒な計算/換算/見積/手数料/按分/逆算）: $2

要件：
- 公式の計算式の出典URLを必ず探す（無ければユーザー入力前提に切る）。
- 想定キーワードとアフィリ/課金先（在庫）を記録。
- 抽象カテゴリは不可。具体的な「○○の××」にする。
- 規制（投資助言/税務代行/医療保証）に触れるなら計算/記録に切り直す。
出力：作成した issue ファイルのパスとねらい（1〜2行）。
