# ユニットのアナトミー（公開可否の最小要件 = spec §5）

公開可能なツールは以下を**すべて**満たす：
1. 動く決定論ロジック（純関数）＋単体テスト（境界値・既知ケース）。
2. 計算式の公式出典（meta.json.formula_sources）。
3. 入力UIと結果表示（CLS を出さない＝寸法予約）。
4. 固有の解説（問題提起→式→手順）。薄い流用は不可。
5. 固有FAQ（3問以上）。
6. 構造化データ（WebApplication + FAQPage [+ HowTo]）。
7. 内部リンク（同カテゴリ）。
8. 広告枠/アフィリ枠（予約）＋PR表記。

doorway page（語句差し替え量産）との差＝「実際に動く固有の機能＋固有の実質的価値」。
