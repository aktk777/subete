# 規制ガードレール（機械的に避ける）

詳細：`references/regulatory-guardrails.md`。

## 判定（1つでも該当＝FAIL or 設計変更）
- 投資/株式：発注する・特定銘柄の売買助言をする・有償で買いシグナルを配信する → NG。
  → **計算・管理・記録・通知**に留める。誰でも買えるサブスクでの情報提供のみ。
- 会計/税務：税理士の独占業務（税務代理・申告書類作成・税務相談）→ NG。
  → 集計・記録・データ出力に留め「申告書を作る/申請を代行する」と謳わない。
- 行政手続き：行政書士の独占業務（許認可申請の代理・書類作成）→ NG。
- 法令対応（電帳法等）の**要件充足の保証**→ NG（誤りが顧客の違反に直結）。
- 医療/法律など人の不利益に直結する領域の**保証・断定**→ NG。

## 表現
- 「概算」「最終判断は各自・専門家へ」を明記。断定（必ず/絶対/確実）を避ける。
