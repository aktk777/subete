# ツール仕様化（実装前に確定する）

`references/spec-template.md` をコピーして埋める。確定すべき最小項目：

1. **slug / title**（kebab-case / 検索意図に合う日本語名）
2. **入力**（各フィールド名・型・単位・既定値・バリデーション）
3. **出力**（各結果・単位・丸め方）
4. **計算式**（厳密に。公式が出している率/式に基づく）
5. **公式出典 URL**（必須。なければ"ユーザー入力前提"に切る）
6. **想定キーワード**（主・副）
7. **レール**（B=広告/アフィリ / A=有料 / B(A)）と**アフィリ/課金先**
8. **FAQ 案**（3問以上・その業務固有）
9. **規制チェック**（投資=助言にしない・税務/行政の独占業務に触れない 等。詳細は jp-compliance）

確定できない（式が公開されていない/事後変動）ものは、その変数を**ユーザー入力**にして決定論を保つ。
